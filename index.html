ngValidationSummary

Angular directive for providing a validation summary to angular based forms.

Download this project as a .zip file Download this project as a tar.gz file

Getting Started

  1. Add references to AngularJS
  2. Add references to ngValidationSummary's js and css files
  3. Where you declare your app module, add ngValidationSummary:
    4. angular.module('myApp',[ngValidationSummary]);
  4. Wrap your html form with the directive ng-validations-container.
    5. <div ng-validations-container>
  <form name="userForm">
    <label for="userName">Name:</label>
    <input id="userName"
           type="text"
           ng-model="model.name"/>
  </form>
</div>
  5. Now each input in the form you want to validate must contain a ng-validation-bubble wich will notify the non-passing validations to the ng-validations-container. an AngularJS or a custom validation (in this example we'll use ng-required) and a validation-friendly-name parameter.
    6. <div ng-validations-container>
  <form name="userForm">
    <label for="userName">Name:</label>
    <input id="userName"
           type="text"
           ng-model="model.name"/>
           ng-required="true"
           ng-validation-bubble=""
           validation-friendly-name="User Name" 
  </form>
</div>
  6. In your html file, within the controller where you plan to use ng-validation-summary, add a div element to contain the UI-listed validations:
    7. <div ng-validation-summary=""></div>
  • Note: This Angular component has no dependecies with jQuery or Bootstrap.
