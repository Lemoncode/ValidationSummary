ngValidationSummary

Angular directive for providing a validation summary to angular based forms.

Download this project as a .zip file Download this project as a tar.gz file

Getting Started

  1. Add references to AngularJS
  2. Add references to ngValidationSummary's js and css files
  3. Where you declare your app module, add ngValidationSummary:
    4. angular.module('myApp',[ngValidationSummary]);
  4. Wrap your html form with the directive ng-validations-container:
    5. <div ng-validations-container>
  <form name="userForm">
    <label for="userName">Name:</label>
    <input id="userName"
           type="text"
           ng-model="model.name"/>
  </form>
</div>
  5. Now, each input in the form you want to validate must contain a ng-validation-bubble wich will notify the non-passing validations to the ng-validations-container. It also must have a supported validation for the input (in this example we use ng-required) and also define a validation-friendly-name
    6. <div ng-validations-container>
  <form name="userForm">
    <label for="userName">Name:</label>
    <input id="userName"
           type="text"
           ng-model="model.name"/>
           ng-required="true"
           ng-validation-bubble=""
           validation-friendly-name="User Name" 
  </form>
</div>
  6. In your html file, within the controller where you want to use, add a div element with theng-validation-summary directive. This div will show the validation results to the user interface :
    7. <div ng-validation-summary=""></div>

And we're done! Now the validation summary will notify the user when the input it's empty.This is the final code and result:

In this simple tutorial, the validation message, "this field is mandatory", is the associated message to the angular ng-required directive. Check out more advanced samples to see how to set your custom messages and custom directives .

Note: This Angular component has no dependecies with jQuery or Bootstrap.
